# Super-admin guide

## Purpose

Manage multiple systematic reviews from one account.

## Sign in

Open `/admin` and sign in with super-admin credentials (created via `npm run bootstrap`).

## Create a review

1. Enter **Review name** and **slug** (lowercase, hyphens allowed)
2. Click **Create**
3. **Bootstrap review admin** — set username/password for the study admin of that review
4. Review admin signs in at `/admin/{slug}`
