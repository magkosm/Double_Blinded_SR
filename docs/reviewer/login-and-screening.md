# Reviewer login and screening

## Purpose

Sign in and screen papers for your systematic review.

## Prerequisites

- Review URL from admin (e.g. `https://…/Double_Blinded_SR/r/my-review`)
- Username and password (one-time credentials from admin)

## Steps

1. Open the review URL
2. Enter username and password → **Sign in**

![Reviewer login](../images/v0.5/reviewer-login.png)

3. Read the rubric panel (tap to expand on mobile)
4. Swipe or tap buttons on each paper card
5. Progress updates in the header; use **Undo last** if needed

![Reviewer card](../images/v0.5/reviewer-card.png)

## When finished

When all papers are decided, you see **All done!**

## Troubleshooting

| Problem | Fix |
|---------|-----|
| Signed in as admin | Sign out admin first or use incognito |
| Loading papers forever | Admin may not have uploaded RIS yet |
| Decrypt error | Contact admin to recreate your account |
