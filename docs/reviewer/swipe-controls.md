# Swipe controls

| Action | Mobile gesture | Desktop |
|--------|----------------|---------|
| **Include (Yes)** | Swipe right | → arrow |
| **Exclude (No)** | Swipe left | ← arrow |
| **Maybe** | Swipe down | ↓ arrow |
| **Skip** | Swipe up | ↑ arrow |

Tap the labeled buttons at the bottom of the card if gestures are awkward.

A colored overlay shows the pending decision while dragging.

## Always visible

**Journal** and **year** (when available) are always shown on the card.

In **title-only** mode (if configured), the abstract is hidden.
